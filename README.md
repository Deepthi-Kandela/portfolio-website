# Personal Portfolio Website

📌 Project Overview

This project is a **Personal Portfolio Website** developed as part of the **HTML/CSS Developer (Front-End Developer) Trainee Program**. The website showcases personal information, projects, and contact details in a clean and responsive layout.

The project demonstrates the use of **HTML for structure** and **CSS for styling**, along with interactive elements such as hover effects, icons, and a contact form to improve the user experience.



 🎯 Objectives

* Build a structured webpage using **HTML5**
* Design and style the website using **CSS3**
* Create a **responsive layout for different devices**
* Add **interactive elements** like hover effects and icons
* Deploy the website online


 🛠 Technologies Used

* HTML5
* CSS3
* Font Awesome Icons


 ✨ Features

* Responsive navigation bar
* About section describing the developer
* Project showcase section
* Contact form for communication
* Hover effects for buttons and links
* Social media icons
* Clean and professional layout
