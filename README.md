# Deepthi Kandela - Frontend Developer Portfolio

 📌 Project Overview
This is my **Personal Portfolio Website**, created as part of the **HTML/CSS Developer (Front-End Developer) Trainee Program**.  
It demonstrates my skills in **HTML, CSS, responsive design, Flexbox, Grid**, and interactive UI elements.

The website features a **fully responsive landing page** showcasing my projects, skills, and contact form.

---

 🎯 Objectives
- Build a structured webpage using **HTML5**  
- Style pages using **CSS3**, Flexbox, Grid, and media queries  
- Implement a **responsive design** for mobile, tablet, and desktop  
- Include **interactive elements** like hover effects and call-to-action buttons  
- Deploy the portfolio using **GitHub Pages**

---

 🛠 Technologies Used
- HTML5  
- CSS3 (Flexbox, Grid, Media Queries)  
- Google Fonts (Poppins)  
- Font Awesome Icons  
- Git & GitHub

---

 ✨ Features
- Fixed navigation bar with smooth scrolling  
- Hero section with introduction and call-to-action button  
- Description section highlighting a featured project with image and button  
- Project gallery with responsive grid layout  
- Testimonials section using Flexbox  
- Contact form for users to reach out  
- Fully responsive layout across devices  
- Hover effects on buttons and project cards  
- Footer with social media icons
